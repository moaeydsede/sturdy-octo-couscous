# SweetLab - Store Category Style
يشبه صفحات التصنيفات في المتاجر: قائمة أقسام + شبكة منتجات + صفحة تفاصيل بخيارات (حجم/وزن) + سلة.

## تعديل رقم واتساب
داخل app.js: WHATSAPP_NUMBER بدون +

## تعديل المنتجات
داخل products.json (variants = خيارات المنتج)

## رفع GitHub Pages
ارفع كل الملفات (بدون مجلدات) ثم Settings > Pages > main/root
