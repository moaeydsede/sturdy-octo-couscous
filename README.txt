# SweetLab Menu Demo (GitHub Pages)

## الملفات
- index.html
- styles.css
- app.js
- products.json

## تشغيل محليًا
افتح `index.html` مباشرة، أو استخدم أي سيرفر بسيط.

## رفع على GitHub Pages (من الموبايل)
1) افتح GitHub > New repository
2) ارفع الملفات الأربعة (بدون مجلدات)
3) Settings > Pages
4) Source: Deploy from a branch
5) Branch: main / (root)
6) سيظهر رابط الموقع بعد الحفظ.

## تعديل رقم واتساب
افتح `app.js` وعدّل:
`WHATSAPP_NUMBER = "201000000000"`
بدون علامة +.

## تعديل المنتجات
افتح `products.json` وعدّل الاسم/السعر/التصنيف/الصورة.
